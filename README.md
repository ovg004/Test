# Joc de la nau

El joc et permet simular el controlament d'una nau en els eixos de X i Y.

## Requeriments

Per jugar a aquest joc has de tenir el programa Python instal·lat.
El Python és pot descarregar a través de la seva pàgina web:

https://www.python.org/downloads/

## Executar el joc

Per executar el joc s'ha d'escriure la seguent comanda:

...
python Nau.py
...
